Replace URL WordPress

Altere facilmente URL's no banco de dados do WordPress!.

Download

Quando migramos um site em WordPress do ambiente de desenvolvimento para o de produção ou vice-versa, sabemos que é um pouco chato alterar os URL's no banco de dados do WordPress usando plugins ou até mesmo SQL. Este script foi criado com intuíto de acabar de vez com essa chatisse utilizando apenas um clique.

Como Usar

  1. Primeiro de tudo, faça o Backup do Banco de dados.
  2. Faça o Upload desse script replace-url.php na raiz do seu host.
    (mesma pasta do arquivo wp-config.php)
  3. Acesse o script diretamente do seu navegador (ex: dominio.com/replace-url.php)
  4. Siga as instruções e clique no botão Salvar.
  5. Delete o script após o uso.
